# これなに
Python3製のクロスコンパイラビルドツールです。
GCC5.3な感じのクロスコンパイラが大量に生成されます。
また開発途中なので色々とお許しください。

# 使い方
まずは`cross_mod/config.py`の中身を参照して`install_dir`の中身に相当するディレクトリを作成し、
一般ユーザーから書き込めるパーミッションを設定してください。
たぶん以下のコマンドでいけるはずです。

```sh
$ sudo mkdir /usr/local/cross
$ sudo chmod 777 /usr/local/cross
```

続いて`cross.py`をPython3.3以上の環境で実行してください。
今のところ`cross.py`の中身は各種モジュールをimportして、
適当にクラスのメソッドを実行するだけのファイルです。
不要なメソッドが動かないように各自コメントを入れるなどして調節してください。
